[//]: # (## How the Demo Works)

[//]: # ()
[//]: # (MasteringX uses a **purchase-first flow**. This means:)

[//]: # ()
[//]: # (1. **User clicks “Buy Course”**)

[//]: # (2. They're redirected to Stripe to complete payment)

[//]: # (3. On successful payment:)

[//]: # (   - The user is **created automatically**)

[//]: # (   - A **random password** is generated)

[//]: # (   - The user is **logged in instantly**)

[//]: # (4. Login credentials are also **sent to the user's email**)

[//]: # ()
[//]: # (> 🧠 There is **no registration form**. Registration happens after a successful payment.)
