## Stripe Integration

You can enable and manage Stripe in two ways:

###  Using the `.env` File

Add the following variables to your `.env` file:

```env
STRIPE_SECRET_KEY=your_stripe_secret_key
STRIPE_PUBLIC_KEY=your_stripe_public_key
````

Be sure to replace the placeholders with your actual Stripe credentials, which are available from your [Stripe dashboard](https://dashboard.stripe.com/).

---

###  Using the Admin Dashboard

You can also enable and configure Stripe directly in the admin panel:

**Admin Dashboard â†’ Settings â†’ Payment Processors**

Changes made through the dashboard are applied instantly.
